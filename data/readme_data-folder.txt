#What's what?
1) collated_summary_data.csv is the main dataset of this project
2) raw-source-data is a .zip file with the raw data that was collated into the file collated_summary_data.csv, see the file 01_collate_summary_data.r in the code folder.
3) the files power_simulation_results.RDS and supplementB_power_simulation_results.RDS are R-data-files of the results of the power simulations. These can be reproduced with the file 02_power_simulation.r in the code folder.
